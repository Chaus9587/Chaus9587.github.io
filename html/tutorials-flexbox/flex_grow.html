Flexbox - Controlling the width of flex items

CSS "flex-grow" property

flex-grow/flex-shrink are used to set the width ratio of flex items.


To make all flex items stretch through the whole flex container in an equal length:

CSS:

.flex-item {
    flex-grow: 1;
}

Output:


More examples below.

CSS: (flex-grow)

.flex-item {
    flex-grow: 1;
}
.flex-item:nth-child(2) {
    flex-grow: 2;
}
.flex-item:nth-child(4) {
    flex-grow: 3;
}

Output:


Syntax

flex-grow: ('number');

flex-grow: inherit;
flex-grow: initial;
flex-grow: unset;
