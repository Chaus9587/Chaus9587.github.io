Flexbox - Aligning multiple flex items

CSS "justify-content" property

justify-content is used to align flex items horizontally.
Take a look at the examples and the respective output below.


We will use the flex container and it's style properties that we just created in the "How to set up a Flexbox container" section.


This is the default properties of justify-content.

CSS: (start/flex-start)

.flex-container {
    justify-content: start/flex-start;
}

Output:


CSS: (end/flex-end)

.flex-container {
    justify-content: end/flex-end;
}

Output:


CSS: (space-between)

.flex-container {
    justify-content: space-between;
}

Output:


CSS: (space-around)

.flex-container {
    justify-content: space-around;
}

Output:


CSS: (space-evenly)

.flex-container {
    justify-content: space-evenly;
}

Output:


CSS: (center)

.flex-container {
    justify-content: center;
}

Output:
