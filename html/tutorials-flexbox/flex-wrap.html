Flexbox - Flex direction and wrapping of flex items

CSS "flex-wrap" property

flex-wrap determines the wrap direction when the flex container size is changing.


Adjust the pop-up window size if necessary to see the effect.

CSS: (nowrap) (default)

.flex-container {
    flex-direction: row;
    flex-wrap: nowrap;
}


CSS: (wrap)

.flex-container {
    flex-direction: row;
    flex-wrap: wrap;
}


CSS: (wrap-reverse)

.flex-container {
    flex-direction: row;
    flex-wrap: wrap-reverse;
}


Syntax

flex-wrap: nowrap;
flex-wrap: wrap;
flex-wrap: wrap-reverse;

flex-wrap: inherit;
flex-wrap: initial;
flex-wrap: unset;
