Flexbox - Controlling the width of flex items

CSS "flex-shrink" property

flex-grow/flex-shrink are used to set the width ratio of flex items. Note that the flex-shrink will only work when the total length of flex items are long enough to overflow the flex container itself.


CSS:

.flex-item {
    flex-basis: 40%;
    flex-shrink: 1;
}
.flex-item:nth-child(2) {
    flex-shrink: 2;
}
.flex-item:nth-child(4) {
    flex-shrink: 3;
}

Output:


4 containers, each having flex-basis of "40%" will overflow the flex container. (160% > 100%)

Syntax

flex-shrink: ('number');

flex-shrink: inherit;
flex-shrink: initial;
flex-shrink: unset;
