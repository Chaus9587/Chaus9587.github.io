Flexbox - Aligning specific flex items

CSS "align-self" property

In some cases, developers may want to specify the location of a flex item. Which align-self can be used in this case. They will overwrite align-items.


Some examples of align-self. We will continue to use the CSS properties of the flex-contatiner and flex-items in the "How to set up a Flexbox container" section.


CSS:

.flex-container {
    align-items: center;
}
.flex-item:nth-child(2) {
    align-self: flex-start;
}

Output:


CSS:

.flex-container {
    align-items: flex-start;
}
.flex-item:nth-child(2) {
    align-self: flex-end;
}

Output:


Syntax

align-self: center;
align-self: start;
align-self: end;
align-self: flex-start;
align-self: flex-end;

align-self: baseline;
align-self: first baseline;
align-self: last baseline;

align-self: inherit;
align-self: initial;
align-self: unset;
